# Carpet Grid Unity Demo

这是从 `C:\Users\jiayu\carpet-grid-demo` 移植出的 Unity 2D/UI 版原型。

## 打开方式

1. 用 Unity Hub 打开 `D:\DEMO`。
2. 打开 `Assets/Scenes/Main.unity`。
3. 点击 Play。游戏会在运行时自动生成 Canvas、棋盘和左右控制栏。

## 已移植功能

- 编辑模式 / 游玩模式，切换模式时默认不进入任何关卡。
- 1 到 99 关，未保存关卡默认无数据。
- 关卡保存到 Unity `PlayerPrefs`。
- 棋盘宽高 1 到 99。
- 地毯数量、长度、起点、目标、组号、可通过异色可编辑。
- 地毯和目标共享 ID，左侧列表可删除。
- 初始格染色。
- 同色借道、指定异色免费通过、异色不可覆盖。
- 同组地毯同步尝试移动，被挡的停下，未被挡的正常移动。
- 铺完后不消失，可沿最近路径撤回。
- 撤回时检测借道依赖，避免破坏其他地毯回退路径。
- 所有地毯长度归零且在各自目标时胜利；游玩模式胜利后自动尝试进入下一关。
- 不再判定失败，可重开当前关。

## 操作差异

网页版本使用左侧工具拖入棋盘；Unity 版本目前采用更稳定的编辑操作：

- 选择颜色和长度。
- 点“放置地毯”或“放置目标”。
- 点击棋盘格放置。
- 在棋盘内按住地毯并拖过相邻格，按玩法规则移动。

## 主要脚本

- `Assets/Scripts/CarpetGridGame.cs`

该脚本自包含运行时 UI、关卡存储和玩法规则，后续可以拆成 `LevelStore`、`BoardView`、`MoveResolver` 等模块。
